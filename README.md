# PUBLIX | Редакционный узел и архив периодических изданий

**PUBLIX** — это многофункциональная веб-система, объединяющая архив периодических изданий (на базе CRUD-операций с JSON-файлами) и интерактивные редакционные чаты. Проект разработан для автоматизации работы редакций и удобного управления публикациями.

---

## Технические характеристики и стек

Проект построен на базе современных веб-технологий и обеспечивает высокую отзывчивость интерфейса.

* **Серверная часть:** Node.js, Express, Socket.IO (для работы чатов в реальном времени).
* **Работа с данными:** CRUD-механизм с использованием постоянного хранилища на базе JSON.
* **Клиентская часть:** HTML5, CSS3 с использованием переменных и адаптивной версткой, JavaScript (ES6+).
* **Стилизация:** Интеграция шрифтов *Playfair Display* и *Caveat* для создания уникального винтажного стиля печатных изданий.

---

## Основной функционал

### 1. Архив периодических изданий (CRUD)
* **Create (Создание):** Добавление новых изданий и статей в базу.
* **Read (Чтение):** Просмотр списка литературы и детальной информации.
* **Update (Редактирование):** Изменение существующих данных.
* **Delete (Удаление):** Снятие публикаций с архива.

### 2. Редакционные узлы (Чат-система)
* **Разделение по комнатам:** Сотрудники могут общаться в разных комнатах (отделах).
* **Ролевая модель:**
  * **Корректор (User):** Может отправлять сообщения и использовать чат.
  * **Редактор (Admin):** Обладает расширенными правами, включая управление комнатами, удаление сообщений, блокировку пользователей (бан) и полную очистку истории.
* **События в реальном времени:** Подключение через WebSockets (Socket.IO).
